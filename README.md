Title: CSS Enhancement of Resume Website

I improved my personal resume website using various CSS concepts learned in the module.

I applied a consistent pastel blue and white color palette to create a clean and professional look. I used different CSS selectors such as element selectors, class selectors, and ID selectors to organize styling efficiently.

I structured the layout using the display property, especially inline block, to arrange skills and project sections in a card format. I also improved spacing and alignment using margin and padding for better readability.
.

Overall, these improvements made the website more visually appealing, structured, and user friendly.
